# 🤖 Automação de Bloco de Notas com PyAutoGUI

Este repositório contém um script simples em Python para demonstrar a automação de interface gráfica (GUI). O script interage com o sistema operacional Windows para abrir o Bloco de Notas e digitar uma mensagem automaticamente.

## 📌 Funcionalidades
* Abre o menu Iniciar do Windows automaticamente.
* Pesquisa e executa o **Bloco de Notas**.
* Realiza a digitação de texto com intervalos configuráveis.
* Inclui travas de segurança para interrupção imediata.

## 🛠️ Pré-requisitos
Antes de rodar o código, você precisa ter o Python instalado e a biblioteca `pyautogui`. Instale-a via terminal:
